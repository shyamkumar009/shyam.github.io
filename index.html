Recent Projects

Mango Detection using SSD MobileNetV2

Framework: TensorFlow

GPU: 1.14

The project is built by training a Single Shot Detector MobileNetV2 model on a dataset of mangoes obtained from Google Images. The system can use any camera pointed towards mango trees to detect mangoes in real-time. The model is trained to draw bounding boxes around the mangoes and count them. The count is displayed as a label on the screen.

Fire Fighting Robot

Platform: Arduino Uno

The robot uses a flame sensor to detect fire and sends the information to an Arduino, which acts as the brain of the robot. Based on sensor input, the Arduino commands the motors to move in the required direction. If the left sensor detects fire, the robot moves left, and the same applies to the front and right sides. The robot stops near the fire source and initiates a watering mechanism until the fire is under control.
